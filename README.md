Energy management and forecasting tools for renewable energy systems with battery storage.

## What's Included

This repository contains code for PV generation forecasting, battery optimization with MPC, and climate scenario analysis for renewable energy systems.

- `RF20_D34_Code_Base.ipynb` - Main Jupyter notebook with all three modules
- `requirements.txt` - Python package dependencies
- `Data/` - Input data folder (PV and climate data)

## Installation

**Requirements:** Python 3.9 or higher
